# servidoresDeRede
1ª Avaliação - Sistema de armazenamento de estado

## Tecnologias
- Vagrant
- VirtualBox
- Shell Script

## Especificações da VM
- Ubuntu
- Apache2

## Setup (Vagrantfile)

```sh
 sudo apt install virtualbox vagrant
```

```sh
 vagrant up
```

```sh
 vagrant ssh
```

## Execução do arquivo main

```sh

 chmod +x main.sh
```
## Depois da permissão concedida, executar o script `main.sh`
```sh
 ./main.sh
```
